# Contact Management System in C

This is a simple contact management system developed using C programming language and linked list data structure.

## Features
- Add contact
- Delete contact
- Search contact
- Display all contacts

## Concepts Used
- Linked List
- Dynamic Memory Allocation
- Pointers

## How to Run

1. Compile:
   gcc miniproject.c -o contact

2. Run:
   ./contact
